## What it is

CPU pixel-level image manipulation — read/generate/transform raw RGBA pixel buffers (`ImageSource`/`Surface`). This is the OpenFL `BitmapData` lineage plus the wider feature set of a software raster library (think `BitmapData` + a slice of ImageMagick/`Pixman`/`tiny-skia`'s raster ops).
