## What it is

Property tweening / interpolation animation — the engine that drives numeric (and color) properties of a target object from a start value to an end value over time, with easing, delay, repeat, yoyo, callbacks, and a manager that advances many tweens per frame. The canonical reference set is GreenSock/GSAP, TweenJS, anime.js, and the OpenFL-adjacent Actuate/Tweener/Tweensy family this SDK draws on.
