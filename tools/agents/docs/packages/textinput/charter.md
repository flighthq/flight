## What it is

Editable-text-field behavior — caret/selection model, text insertion/deletion/replacement, keyboard and pointer editing commands, input restrictions, password masking, and the focus/dispatch managers that wire raw input into one focused field. The OpenFL/Flash `TextField` "input" mode (TextFieldType.INPUT) plus a read-only "selectable text" mode, expressed as free functions over a `RichText` entity and its runtime input slot.
